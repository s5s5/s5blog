## [](http://s5s5.github.io/Code-snippets-of-s5s5/#html)HTML

HTML文档声明

```html
<!DOCTYPE html>
```

HTML编码使用utf-8，放置于

区最上方 （文件保存时存为 utf-8 无BOM）

```html
<meta charset="utf-8" />
```

强制IE系浏览器用最新渲染模式渲染

```html
<meta http-equiv="X-UA-Compatible" content="edge" />
```

多终端设备访问时自动缩放到屏幕宽度

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```

Meta 值是SEO优化的关键

```html
<meta name="description" content="" /> <meta name="keywords" content="" />
```

Title 部分更是SEO优化的关键

```html
<title></title>
```

禁止使用 \* 选择器

```css
* {
  margin: 0;
}
```

禁止使用expression

```css
.some_element {
  width: expression((documentElement.clientWidth < 725) ? "725px": "auto");
}
```

最终发布样式避免使用[@import](https://github.com/import)

```css
@import url("a.css");
```

尽量避免使用 ID 选择器，为了提升样式重用性以及与页面的耦合性

```css
#id {
  margin: 0;
}
```

尽量避免使用 !important，为了减少后续工作复杂度

```css
.some_element {
  color: red !important;
}
```

禁用 outline:none 或 outline:0，为了帮助使用我们产品有障碍的人士（如：盲人）

```css
a {
  outline: none;
}
a {
  outline: 0;
}
```

值为0时不需要任何单位

```css
.some_element {
  padding: 0 5px;
}
```

避免使用多类选择符，IE6以及更古老的浏览器对类似 .some.element 的多类选择符解析不正确

```css
.some.element {
  color: red;
}
```

移除空的css规则

```css
.some_element {
}
```

正确使用display的属性，由于display的作用，某些样式组合会无效，徒增样式体积的同时也影响解析性能

```css
.some_element {
    display:inline //后不应该再使用 width 、 height 、 margin 、 margin-top 、 margin-bottom 以及 float
    display:inline-block //后不应该再使用 float
    display:block //后不应该再使用 vertical-align
    display:table-* //后不应该再使用 margin 或者 float
}
```

隐藏文字用text-indent: -999方法时，要指定单位和文字方向

```css
.some_element {
  direction: ltr;
  text-indent: -999em;
}
```

filter：应该尽量避免使用 AlphaImageLoader，可以适当在投影、发光、渐变、去色方面上使用

```css
.some_element {
  background: url(image.png);
  _background: none;
  _filter: progid:DXImageTransform.Microsoft.AlphaImageLoader(src='image.png', sizingMethod='crop');
}
```

禁止重复属性

```css
.some_element {
  width: 100px;
  width: 120px;
}
```

不滥用web字体，web fonts通常体积庞大，而且一些浏览器在下载web fonts时会阻塞页面渲染损伤性能。

```css
@media screen {
  @font-face {
    font-family: "Flavors";
    font-style: normal;
    font-weight: 400;
    src:
      local("Flavors"),
      local("Flavors-Regular"),
      url("http://themes.googleusercontent.com/static/fonts/flavors/v1/7ed-D3MQicPNqc0ZKJ1T1Q.woff")
        format("woff");
  }
}
```

标准化各种浏览器前缀，将浏览器前缀置于前面，将标准样式属性置于最后

```css
.some_element {
  -webkit-box-shadow: 4px 5px 6px #789;
  -moz-box-shadow: 4px 5px 6px #789;
  box-shadow: 4px 5px 6px #789;
}
```

使用CSS渐变等高级特性，需指定所有浏览器的前缀

```css
.some_element {
  background: -moz-linear-gradient(top, #1e5799 0%, #7db9e8 100%);
  background: -webkit-gradient(
    linear,
    left top,
    left bottom,
    color-stop(0%, #1e5799),
    color-stop(100%, #7db9e8)
  );
  background: -webkit-linear-gradient(top, #1e5799 0%, #7db9e8 100%);
  background: -ms-linear-gradient(top, #1e5799 0%, #7db9e8 100%);
  background: -o-linear-gradient(top, #1e5799 0%, #7db9e8 100%);
}
```

rgba(), hsl(), hsla()只现代浏览器支持，使用时还要加上备用色

```css
.some_element {
  color: red;
  color: rgba(255, 0, 0, 0.5);
}
```

使用CSS3圆角时，添加background-clip，以防止某些浏览器显示不正常

```css
.some_element {
  -webkit-border-radius: 12px;
  -moz-border-radius: 12px;
  border-radius: 12px;
  -moz-background-clip: padding;
  -webkit-background-clip: padding-box;
  background-clip: padding-box;
}
```


自定义鼠标路径要用绝对路径，不然不显示

