# BLOG

This is my BLOG，基于 [Astro](https://astro.build/) 构建，内容涵盖技术分享、生活随笔与学习记录。

## 项目亮点

- 轻量、极致性能，支持 Markdown/MDX
- 支持 SEO、RSS、Sitemap
- 评论系统自建，数据安全可控
- 响应式设计，移动端友好

## 快速开始

1. 安装依赖：

   ```sh
   npm install
   ```

2. 本地开发：

   ```sh
   npm run dev
   ```

3. 构建生产环境：

   ```sh
   npm run build
   ```

4. 预览构建结果：

   ```sh
   npm run preview
   ```

## 目录结构

```text
├── public/           # 静态资源
├── src/
│   ├── components/   # 组件
│   ├── content/      # 博客内容（Markdown/MDX）
│   ├── layouts/      # 布局
│   └── pages/        # 路由页面
├── astro.config.mjs  # Astro 配置
├── package.json      # 项目依赖
├── tsconfig.json     # TypeScript 配置
```
